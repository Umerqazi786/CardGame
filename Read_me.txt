Card game project is written in python programming language. To run the card game file, go to the directory containing Qazi_Umer.py file in command prompt. 
File can be run with command "python Qazi_Umer.py". Once the file is running, user can select different game modes by providing keyboard input for number of players.
Python 3.0 or later versions are better to run Qazi_Umer.py file.
